本地仓库ssh连接多个远程仓库

由于GitHub不同的两个账户不可以配置相同的公钥，因此本地仓库ssh连接多个远程仓库必须生成不同的密钥

1.生成密钥

对于使用ssh连接远程仓库，需要生成非对称密钥并将公钥添加到远程仓库中

通常id_rsa和id_rsa.pub是默认密钥，额外生成的密钥需要自定义

2.配置ssh的config

  1. 创建config文件

  2. 编辑配置文件，使不同密钥指向不同远程仓库

  3. 清空本地缓存，添加新的ssh密钥到SSH agenth中

  4. 测试ssh链接

3. 仓库配置
